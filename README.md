# Inventory Management System

This project is based on SmartShelfX inventory system. It is an AI-based system for inventory forecasting and stock management.

## Features
- Inventory tracking
- Stock management
- Role-based access (Admin, Vendor, Client)
- Inventory prediction using Machine Learning
